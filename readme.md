# SIMPLE LOGIN APP

## Common Setup

Clone the repo and install the dependencies.

```
git clone https://github.com/adhimaswaskita/Ping-Data-Collector.git
```
```
npm install
```

## Running The Project

1. Open project folder

2. To start the express server, run 'node app.js'
